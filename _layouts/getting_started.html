getting started

We would like to help you get started. At first, for the purpose of this presentation, we need to include some required packages: 

library(gmum.r)
library(ggplot2)

Clustering data set

Now we might prepare two simple ellipsoids, which act as a data set for further computation: 

dataset <- svm.dataset.2e()

It is good to make the column names be more descriptive and drop them with class labels: 

colnames(dataset) <- c("X", "Y", "Target")
dataset <- dataset[c("X", "Y")]

If we plot this data set, we should see something like this:

Such data set can be easily clustered. We would like to create two clusters from the data set, so we set k to be equal to 2: 

clusters <- CEC(x=dataset, k=2)

The clusters appear in the plot as follows:

Building classifier

The clusters have been created, therefore we may build a classifier. At first, we have to add labels to the data set since they are required for training SVM. We need to add one label in our case: 

target <- clusters$y()
dataset["Target"] = target

Below is a complete set of commands which trains SVM: 

formula <- Target ~ .
svm <- SVM(formula, data=dataset, lib="libsvm", kernel="linear", C=10)

The decision boundary should be as follows:

The SVM has been trained. Next we may test the SVM using some sample data. For this purpose, we may pick a sample matrix: 

test <- matrix(rnorm(200, mean=0, sd=2), , 2)

And run prediction: 

prediction <- predict(svm, test)
df <- data.frame(X=test[,1], Y=test[,2], Target=prediction)
result <- ggplot() + geom_point(data=df, mapping=aes(X,Y), colour=factor(df$Target + 1))
