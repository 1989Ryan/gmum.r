getting started

We would like to help you get started. At first, let's include packages required for this presentation: 

library(gmum.r)
library(ggplot2)

Now we might prepare two simple ellipsoids, which act as a data set for further computation: 

dataset <- svm.dataset.2e()

Let's make the column names be more descriptive and drop them with class labels: 

colnames(dataset) <- c("X", "Y", "Target")
dataset <- dataset[c("X", "Y")]

If we plot this data set, we should see something like this:

Such data set can be easily clustered in the next step. In this example we would like to get two clusters from the data set, so we set k to be equal to 2: 

clusters <- CEC(x=dataset, k=2)

Clusters appear in the plot like this:

We need labels to be able to train SVM, let's add them: 

target <- clusters$y()
dataset['Target'] = target

Now it's a high time we trained SVM: 

formula <- Target ~ .
svm <- SVM(formula, data=dataset, lib='libsvm', kernel='linear', C=10)

The decision boundary should be as follows:

Alright, SVM has been trained. Next we may test our SVM on some sample data. Let's pick a sample matrix: 

test <- matrix(rnorm(200, mean=0, sd=2), , 2)

And run prediction: 

prediction <- predict(svm, test)
df <- data.frame(X=test[,1], Y=test[,2], Target=prediction)
result <- ggplot() + geom_point(data=df, mapping=aes(X,Y), colour=factor(df$Target + 1))
